== Description ==

Only load plugins when necessary and make your WordPress load faster. With this plugin you're allowed to only load plugins based on specific category, page or post.

When installing make sure the mu-plugin folder is writable or else just copy the file under the move_me folder to the mu-plugin folder.

Bug reports and ideas are very welcome!

== Installation ==

1. Activate the plugin through the 'Plugins' menu in WordPress
2. Make sure you've a selective loading file under the mu-plugin folder, or else just copy the file from the move_me fodler to the mu-plugin folder.

== Frequently asked questions ==

= Why doesn't the plugin work =

Please make sure you've the selective loading file from /move_me under the mu-plugin folder.

= Where do I activate/deactivate plugins =

You can activate it under settings -> selective loading and/or on a specific page/post when your are editing.
